# Github Actions - CI/CD Pipelines

<font style='color:orange'>GitHub Actions</font><font style='color:white'> is a powerful automation tool provided by GitHub that enables developers to automate tasks related to their software development workflows. It allows for the creation of custom **workflows**, which can be triggered by various events, such as pushes to a repository, pul requests, or scheduled events. Thsese workflows can be used for various purposes, including <font style='color:darkorange'>Continuous Integration (CI), Continuous Deployment</font>(CD), testing, and more.
